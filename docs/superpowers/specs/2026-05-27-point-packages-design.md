# Design Spec: Point Packages System (Standard & Enterprise)

**Date:** 2026-05-27
**Status:** Approved

## 1. Overview
This feature allows administrators to manage different types of subscription/point packages. It includes "Standard" packages (for direct point purchase) and "Enterprise" packages (for B2B contact).

## 2. Database Schema

### Table: `point_packages`
| Column | Type | Constraints | Description |
| :--- | :--- | :--- | :--- |
| `id` | Integer | Primary Key, Auto-increment | Unique identifier |
| `name` | String | Not Null | Name of the package |
| `type` | String | Not Null | `STANDARD` or `ENTERPRISE` |
| `price` | Float | Nullable | Price for standard packages |
| `points` | Integer | Nullable | Points provided in the package |
| `description` | Text | Nullable | Detailed description |
| `is_active` | Boolean | Not Null, Default: True | Visibility status |
| `created_at` | Timestamp | Not Null | Creation time |
| `updated_at` | Timestamp | Not Null | Last update time |

## 3. Backend API (FastAPI)

### Admin Endpoints (Protected)
*   `GET /api/admin/packages`: List all packages.
*   `POST /api/admin/packages`: Create a new package.
*   `PUT /api/admin/packages/{id}`: Update an existing package.
*   `DELETE /api/admin/packages/{id}`: Delete a package.

### Public Endpoints
*   `GET /api/packages`: List all active packages (for users).
*   `POST /api/contact-enterprise`: Handle enterprise contact form submissions (will log to console/DB for now).

## 4. Frontend Implementation (React/Ant Design)

### Admin Side
*   **New Page:** `AdminPackageManagementPage.tsx` in `src/pages/admin/packages/`.
*   **Component:** A `Table` displaying package details and a `Modal` with a form for adding/editing.

### User Side
*   **New Page:** `PricingPage.tsx` in `src/pages/`.
*   **Components:**
    *   `PackageCard`: Displays package info.
    *   `StandardPackageCard`: Includes "Buy Now" button.
    *   `EnterprisePackageCard`: Includes "Contact Us" button.
*   **Contact Modal:** A popup form for enterprise inquiries.
