# Hvideo Lite — Hướng dẫn Cán bộ

## 1. Kích hoạt tài khoản
- Admin tạo tài khoản → bạn nhận email OTP 6 số
- Mở link trong email (hoặc vào `/verify-otp`), nhập OTP
- Đặt mật khẩu (tối thiểu 8 ký tự)
- Đăng nhập bằng email + mật khẩu

> Nếu OTP hết hạn hoặc sai quá 5 lần, liên hệ Admin.

## 2. Dashboard
- Menu Dashboard → xem số dư point, số lượt xử lý, biểu đồ 7 ngày, 5 giao dịch gần nhất

## 3. Phục hồi biển số (M01)
- Menu Biển số
- **Từ ảnh:** Tải ảnh → chỉnh sửa (sáng/tương phản/bão hòa/xoay/lật/zoom) → chọn quốc gia/loại xe/màu biển → nhấn Xử lý
- **Từ video:** Chuyển sang tab Video → tải video → chọn frame → cấu hình → Xử lý
- Kết quả: biển số + độ tin cậy (%)

## 4. Sửa video hỏng (M02)
- Menu Sửa video
- Tải video hỏng → nút "Phân tích lỗi"
- Chọn chế độ: Nhanh (10 PT, ~2 phút) / Sâu (20 PT, ~8 phút)
- Tùy chọn codec, mức sửa (1-5), giữ audio
- Nhấn "Bắt đầu sửa"
- Thành công: xem chi tiết lỗi đã sửa. Thất bại: không trừ point

## 5. Lịch sử giao dịch
- Menu Giao dịch → xem lịch sử point của bạn
- Lọc theo dịch vụ, xuất CSV

## 6. File
- Menu File → quản lý file cá nhân
- Tải lên (ảnh/video), tải xuống, xóa

## 7. Hồ sơ
- Menu Hồ sơ → xem/sửa thông tin, đổi mật khẩu
