Luồng hoạt động

  • Link: Click "Liên kết" → popup mock OAuth → lưu provider ID vào user → ghi log

  • Login bằng OAuth đã link: Tìm user có provider ID → login thẳng
  • Hủy liên kết cuối cùng: Nếu chỉ còn 1 cách đăng nhập → cảnh báo
