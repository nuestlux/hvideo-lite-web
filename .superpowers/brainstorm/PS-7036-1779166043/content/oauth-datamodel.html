Giải thích flow OAuth Linking

  1. Login bằng OAuth (chưa có account): → Tạo user mới + lưu oauth_links + audit_log
  2. Login bằng OAuth (đã có account): → Tìm user có oauth_links.{provider} = id → login thẳng
  3. Link OAuth từ Profile: → User đã login → chọn "Liên kết" với provider → lưu vào oauth_links
  4. Unlink OAuth từ Profile: → Xóa provider entry → cảnh báo nếu không còn cách login nào khác
