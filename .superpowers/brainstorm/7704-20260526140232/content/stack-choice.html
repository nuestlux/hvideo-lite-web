Chọn Technical Stack cho Hvideo Lite

Phase 1 — Core Platform. Bạn muốn dùng stack nào?

A

Python + SQLite

Backend: FastAPI (Python)
Frontend: React / Vue
Database: SQLite (nhẹ, không cần cài server)

Ưu

  • Đơn giản, dễ triển khai
  • Python phù hợp AI/ML pipeline
  • SQLite không cần quản trị DB

Nhược

  • SQLite không hỗ trợ concurrent cao
  • 500 users có thể chậm
B

Node.js + SQLite

Backend: NestJS / Express (Node.js)
Frontend: React / Vue
Database: SQLite (nhẹ, không cần cài server)

Ưu

  • JavaScript xuyên suốt (FE + BE)
  • SQLite đơn giản
  • Next.js có thể SSR

Nhược

  • Không tối ưu cho AI pipeline (sau này)
  • SQLite concurrent hạn chế
C

Python + PostgreSQL (khuyến nghị)

Backend: FastAPI (Python)
Frontend: React / Vue
Database: PostgreSQL (chuyên nghiệp)

Ưu

  • Phù hợp 500 users + concurrent
  • Python sẵn sàng cho AI Phase 3
  • PostgreSQL ổn định, bảo mật

Nhược

  • Cần cài đặt PostgreSQL server
  • Phức tạp hơn SQLite
